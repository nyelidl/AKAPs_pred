คู่มือการใช้งาน AKAP Domain Screener

สำหรับเว็บไซต์ https://akapspred.streamlit.app/ — อธิบายวิธีใช้โปรแกรมและวิธีแปลผลเป็นภาษาไทยแบบเข้าใจง่าย

เหมาะสำหรับผู้เริ่มต้น อธิบายทีละขั้นตอน มีแนวทางแปลผล ใช้กับชื่อโปรตีนหรือ UniProt ID ได้

โปรแกรมนี้ใช้ทำอะไร?

โปรแกรมนี้ใช้สำหรับ คัดกรองลำดับโปรตีน เพื่อค้นหาว่ามีบริเวณที่มีแนวโน้มเป็น AKAP motif / AKAP domain หรือไม่ โดยจะประเมินว่าบริเวณนั้นคล้าย motif ที่จับกับ PKA regulatory subunit มากน้อยเพียงใด

  • ใช้ค้นหา candidate AKAP จากชื่อโปรตีน หรือ UniProt accession
  • ช่วยแยกผลลัพธ์เป็น AKAP / DDIP / ambiguous / unlikely
  • ช่วยบอกแนวโน้มความจำเพาะต่อ RI หรือ RII

รูปที่ 2: ความหมายของ classification

AKAP

น่าเชื่อถือสูง
เป็น candidate ที่มีลักษณะสอดคล้องกับ AKAP motif มากที่สุด ควรเป็นตัวเลือกแรก ๆ สำหรับการศึกษาต่อ

DDIP

อาจจับ d/d domain แต่ไม่ใช่ PKA anchor จริง
ควรระวังว่าอาจเป็น DDIP มากกว่า AKAP

ambiguous

ก้ำกึ่ง
ยังไม่ชัดเจน ต้องใช้ข้อมูลเพิ่มเติม เช่น โครงสร้าง, conservation, experiment

unlikely

ความเป็นไปได้น้อย
มักมี negative determinant หรือคุณสมบัติไม่สอดคล้องกับ AKAP motif

รูปที่ 3: วิธีเลือก hit ที่ควรสนใจ

  1. เริ่มจากดู classification ก่อน — ถ้าเป็น AKAP จะน่าสนใจที่สุด
  2. ดู pssm_score — เลือก hit ที่มีคะแนนสูงกว่า hit อื่นในโปรตีนเดียวกัน
  3. ดู ml_prob — ถ้าค่าสูงจะช่วยสนับสนุนว่า hit นั้นดี
  4. ดู n_negdet — ถ้าเป็น 0 จะดีกว่า เพราะไม่มี negative determinant
  5. ดู predicted_specificity — ถ้าคุณสนใจ RI หรือ RII เป็นพิเศษ ให้ใช้ค่านี้ช่วยตัดสิน
  6. ดู Detailed View และ Protein map เพื่อเข้าใจตำแหน่งของ motif ในโปรตีน
ตัวอย่างการอ่านผล: ถ้า hit มี classification = AKAP, pssm_score สูง, ml_prob สูง, และ n_negdet = 0 → ถือว่าเป็น candidate ที่ดีมากสำหรับการศึกษาต่อ

ข้อควรระวังในการแปลผล

  • ผลจากโปรแกรมเป็น prediction / screening ไม่ใช่การยืนยันเชิงทดลอง
  • hit ที่ดีควรนำไปตรวจต่อด้วยข้อมูลอื่น เช่น conservation, structure prediction, mutational analysis หรือ biochemical assay
  • ถ้าโปรตีนหนึ่งมีหลาย hit ให้พิจารณาทั้งคะแนน, classification และตำแหน่งในโปรตีนร่วมกัน
